## Install

```js
npm install @citation-js/plugin-isbn
```

## Use

Install the plugin by `require`-ing it:

```js
require('@citation-js/plugin-isbn')
```

## Formats

Formats and other features added by this plugin.

### ISBN identifiers

  * ISBN-10
  * ISBN-13
  * ISBN-A (actionable ISBN)
